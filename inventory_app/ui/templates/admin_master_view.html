Text Rules - Upper/Lower Case Preset Rules Quick guide for save formatting
  • Description / Notes / Restock: Title Case on save; joiner words stay lowercase after the first word.
  • CAPS prompt: if any ALL-CAPS words are found in those text fields, save asks once whether to keep them uppercase.
  • Box: Title Case letters and numbers only, for example Zen Zone 2.
  • Location: forced to full UPPER CASE, for example ZEN ZONE 2.
  • Event Tags: MUST be Typed and stored as uppercase pipe tags like |NEEDED||PIPES|.
  • Measurements: matched units are normalized on save, for example 25 m becomes 25m.
